Kế hoạch thiết kế giao diện — Nền tảng giáo dục
Nền tảng phục vụ 5 nhóm đối tượng, mỗi nhóm có dashboard riêng biệt nhưng dùng chung core modules: bài tập, chấm điểm, theo dõi tiến độ, quản lý khoá học.

3 · Luồng người dùng chính (User flow)

03

04

Auto-grade → GV review → gửi nhận xét
05
Xem báo cáo
Dashboard → drill-down → export
